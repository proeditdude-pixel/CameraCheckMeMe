# CameraCheckMeMe — Spatial Reaction Experience

CameraCheckMeMe is a high-fidelity, web-based AI application that uses computer vision to trigger real-time visual reactions based on your facial expressions and hand gestures. Built with a premium "Spatial" aesthetic, it utilizes MediaPipe's neural networks to track landmarks directly in your browser.

📸 Screenshots

The spatial interface in action, tracking gestures and triggering reactions.

✨ Features

Spatial UI: A premium, Apple-inspired interface with glassmorphism, fluid animations, and a focus on content.

Dual Tracking: Synchronized tracking of facial landmarks and hand poses.

Real-time Reactions: Instant visual feedback (GIFs/Images) triggered by specific physical actions.

Privacy First: All AI processing happens locally on your device; no video data is ever sent to a server.

🛠️ Technology Stack

MediaPipe: Google's high-performance ML framework for face and hand tracking.

Tailwind CSS: For the modern, responsive layout.

JavaScript (ES6+): Logic for gesture thresholds and UI state management.

Google Fonts: Inter font family for a premium feel.
